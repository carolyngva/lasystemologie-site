# lasystemologie-site

Site Astro pour `lasystemologie.com`.

## Commandes

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Déploiement Cloudflare Pages

- Framework preset : Astro
- Build command : `npm run build`
- Build output directory : `dist`
